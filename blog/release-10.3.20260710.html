📋 Changelog​

SHAFT 10.3.20260710

Auto-generated release notes are included below. Users upgrading from SHAFT_ENGINE should read the modular SHAFT upgrade guide.

SHAFT Pilot adds deterministic Capture, TestNG generation, Doctor diagnosis, reviewed repair proposals, and MCP interoperability. AI is optional, disabled by default, and direct OpenAI, Anthropic, Gemini, or Ollama access requires explicit enablement and consent. Microsoft/GitHub Copilot integrates through MCP rather than a generic provider API-key adapter.

See the SHAFT Pilot guide for installation, configuration, privacy, troubleshooting, and usage examples.

What's Changed​

  • Fix scheduled E2E CI failures: shaft-sikulix opencv, stale test expectation, Appium install race by @MohabMohie in https://github.com/ShaftHQ/SHAFT_ENGINE/pull/3408
  • Fix SHAFT Assistant timeouts, approve-all checkbox, approval-box overflow, and recorder stop by @MohabMohie in https://github.com/ShaftHQ/SHAFT_ENGINE/pull/3410
  • SHAFT Assistant: real interactive local-agent approval, richer Verbose detail, fix mid-run toggle by @MohabMohie in https://github.com/ShaftHQ/SHAFT_ENGINE/pull/3411
  • Portable setup, deep Test Connection, verbose passthrough, and recorder/codegen fixes (closes #3409) by @MohabMohie in https://github.com/ShaftHQ/SHAFT_ENGINE/pull/3418

Full Changelog: https://github.com/ShaftHQ/SHAFT_ENGINE/compare/10.3.20260709...10.3.20260710
