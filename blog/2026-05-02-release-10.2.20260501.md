## ⬆️ How to Upgrade

Update the version in your `pom.xml`:

```xml
<properties>
    <shaft_engine.version>10.2.20260501</shaft_engine.version>
</properties>
```

Or update the dependency directly:

```xml
<dependency>
    <groupId>io.github.shafthq</groupId>
    <artifactId>SHAFT_ENGINE</artifactId>
    <version>10.2.20260501</version>
</dependency>
```

> [!IMPORTANT]
> **We support only the latest release.** If you encounter any issue, please upgrade to `10.2.20260501` first before filing a bug report.
