## Report a Bug

1. Check [existing issues](https://github.com/ShaftHQ/SHAFT_ENGINE/issues) first
2. If new, [create an issue](https://github.com/ShaftHQ/SHAFT_ENGINE/issues/new) with:
   - SHAFT version
   - Java version
   - Steps to reproduce
   - Expected vs. actual behavior
   - Stack trace / logs

## Request a Feature

Use [GitHub Discussions](https://github.com/ShaftHQ/SHAFT_ENGINE/discussions) to propose features. The maintainers triage requests and move accepted ones to Issues.

## Contributing

SHAFT welcomes contributions of any size:

- **Bug fixes** — Find and fix issues
- **Documentation** — Improve guides, add examples
- **New features** — Extend platform support
- **Tests** — Increase coverage

See the [contribution guide on GitHub](https://github.com/ShaftHQ/SHAFT_ENGINE/blob/master/CONTRIBUTING.md) for setup instructions.
